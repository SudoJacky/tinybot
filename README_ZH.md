[English](README.md)

Tinybot 是一个由大语言模型和原生工具系统驱动的轻量级个人 AI 助手。

安装 Tinybot 时，请前往 [GitHub Tags](https://github.com/SudoJacky/tinybot/tags)，选择最新版本并下载适合当前平台的安装包。
