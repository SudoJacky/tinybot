# 工具功能

## AI 能做什么？

Tinybot 的 AI 可以使用多种工具来帮你完成任务：

---

## 文件操作

| 功能 | 说明 |
|------|------|
| 读取文件 | 查看文件内容 |
| 编辑文件 | 修改文件内容 |
| 创建文件 | 新建文件 |
| 搜索文件 | 按名称查找文件 |
| 搜索内容 | 在文件中搜索文字 |

**示例：**

```
用户：读取 config.json 文件

AI：[读取文件并显示内容]
```

---

## 执行命令

AI 可以在你的电脑上执行命令：

```
用户：列出当前目录的文件

AI：[执行 ls 命令并显示结果]
```

### 安全限制

可以设置只允许在特定目录操作，防止 AI 修改重要系统文件。

---

## 网络搜索

AI 可以搜索互联网获取最新信息：

```
用户：搜索今天的科技新闻

AI：[搜索并汇总结果]
```

### 搜索引擎选择

| 选择 | 说明 |
|------|------|
| DuckDuckGo | 默认，免费无需配置 |
| Brave | 需要API 密钥 |
| Tavily | 需要 API 密钥 |

### 代理设置

如果网络受限，可以配置代理让 AI 能访问外网。

---

## 网页获取

AI 可以访问网页并提取内容：

```
用户：帮我看看这个网页的内容

AI：[访问网页并总结内容]
```

---

## 定时任务

设置定时提醒和任务：

```
用户：每天早上9点提醒我开会

AI：[创建定时任务]
```

---

## 工具设置

### 启用/禁用工具

可以在配置中控制哪些工具可用：

- 禁用网络搜索 → AI 不会搜索网页
- 禁用命令执行 → AI 不能运行命令

### 安全设置

| 设置 | 说明 |
|------|------|
| 工作区限制 | 限制 AI 只在指定目录操作 |
| 超时时间 | 命令执行的最大时间 |

---

## 高级工具

### MCP 工具

可以连接外部工具服务，扩展 AI 的能力。

### 浏览器自动化

AI 可以操作浏览器：

- 打开网页
- 点击按钮
- 输入文字
- 提取信息

需要额外安装浏览器扩展。

---

## 常见问题

### AI 说没有权限？

检查工作区限制设置，可能目录不在允许范围内。

### 网络搜索失败？

检查网络连接，或配置代理。

### 命令执行超时？

可能是命令运行时间太长，可以增加超时时间设置。
