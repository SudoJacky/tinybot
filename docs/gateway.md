# 网关服务

## 什么是网关？

网关服务让你可以：

- 使用网页界面和 AI 对话
- 连接多个聊天平台（飞书、钉钉等）
- 设置定时任务

---

## 启动网关

```bash
uv run tinybot gateway
```

然后打开浏览器访问 `http://127.0.0.1:18790`

---

## 网页界面设置

### 访问地址

默认地址：`http://127.0.0.1:18790`

### 修改端口

如果默认端口被占用，可以修改配置：

```json
{
  "gateway": {
    "port": 8080
  }
}
```

### 只允许本机访问

使用 `127.0.0.1` 作为地址，只有本机能访问。

### 允许局域网访问

使用 `0.0.0.0` 作为地址，局域网内其他设备可以访问。

---

## 连接聊天平台

网关可以连接到多个聊天平台，让 AI 在这些平台回复消息。

### 飞书

1. 在飞书开放平台创建应用
2. 获取 App ID 和 App Secret
3. 在配置中添加：

```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "app_id": "你的App ID",
      "app_secret": "你的App Secret"
    }
  }
}
```

### 钉钉

类似飞书，需要在钉钉开放平台创建应用。

### 微信企业号

在微信公众平台创建应用后配置。

## 定时任务

网关支持设置定时任务，让 AI 定期执行某些操作。

### 通过对话设置

直接和 AI 说：

```
每天早上9点提醒我检查日程
```

AI 会帮你设置定时任务。

### Cron 表达式

定时任务使用 cron 表达式：

| 表达式 | 含义 |
|--------|------|
| `0 9 * * *` | 每天9点 |
| `0 9 * * 1-5` | 工作日9点 |
| `*/30 * * * *` | 每30分钟 |

---

## 心跳服务

网关空闲时会自动执行一些维护任务：

- 整理对话历史
- 分析和学习经验

这个功能默认开启，可以保持 AI 的长期记忆。

---

## 常见问题

### 网页无法访问？

检查：
1. 网关服务是否启动
2. 端口是否被占用
3.防火墙是否拦截

### 其他平台无法收到消息？

检查：
1. 平台配置是否正确
2. 应用权限是否设置
3. 网关服务是否运行
