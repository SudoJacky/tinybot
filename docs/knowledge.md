# 知识库

## 什么是知识库？

知识库让你的 AI 能够学习和引用你提供的文档内容。比如：

- 你添加了项目说明书，AI 就能回答关于项目的问题
- 你添加了产品文档，AI 就能解答产品相关问题
- 你添加了工作流程文档，AI 就能指导用户操作

---

## 启用知识库

### 方法一：网页界面设置

1. 打开网页界面
2. 点击右侧设置面板
3. 找到 Knowledge 配置
4. 将 `enabled` 设置为开启

### 方法二：修改配置文件

在配置文件中设置：

```json
{
  "knowledge": {
    "enabled": true
  }
}
```

---

## 添加文档

### 通过网页界面

1. 打开网页界面
2. 点击右侧"知识库"面板
3. 点击"添加"按钮
4. 输入文档名称和内容
5. 保存

### 上传文件

支持上传 `.txt` 和 `.md` 文件：

1. 在知识库面板点击"上传"
2. 选择文件
3. 文件内容自动添加到知识库

---

## 使用知识库

启用后，当你提问时：

1. AI 自动搜索相关文档
2. 找到相关内容后引用
3. 结合知识回答你的问题

**示例：**

```
用户：项目的部署流程是什么？

AI：根据项目文档，部署流程如下：
1. 安装依赖
2. 配置环境变量
3. 运行启动命令
...
```

## 知识库设置说明

| 设置 | 说明 | 推荐值 |
|------|------|--------|
| 自动检索 | 对话时自动搜索知识 | 开启 |
| 检索数量 | 每次搜索返回的片段数 | 5 |
| 分块大小 | 文档切分的段落大小 | 500 |

### 检索数量

控制每次搜索返回多少内容片段：
- 数值越小，返回内容越少，响应更快
- 数值越大，返回内容越多，信息更全面

### 分块大小

文档会被切分成小块存储：
- 小块适合精确匹配
- 大块保留更多上下文

---

## 检索模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 混合模式 | 默认模式，结合语义和关键词 | 大多数情况 |
| 语义模式 | 按意思相似度搜索 | 内容表达多样的文档 |
| 关键词模式 | 按关键词精确匹配 | 技术文档、代码 |

## 高级功能

### 重排序

开启后可以提高搜索准确性，但需要额外的 API 服务。

**是否需要？**
- 一般使用不需要开启
- 如果搜索结果不准确，可以尝试开启

### 文档分类

添加文档时可以设置分类和标签，方便管理：

- 分类：如"产品文档"、"技术文档"
- 标签：如"入门"、"高级"、"API"

---

## 管理知识库

### 查看文档列表

在网页界面的知识库面板可以看到所有已添加的文档。

### 删除文档

点击文档旁的删除按钮即可删除。

### 更新内容

文档更新后，点击"重建索引"让知识库重新处理。

---

## 常见问题

### AI 没用到我的文档？

检查：
1. 知识库是否已启用
2. 问题是否和文档内容相关
3. 尝试重建索引

### 搜索结果不准确？

尝试：
1. 调整检索数量
2. 更换检索模式
3. 开启重排序功能
