# 命令行界面

## 启动聊天

在终端运行：

```bash
uv run tinybot agent
```

进入聊天界面后，直接输入文字和 AI 对话。

---

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 发送消息 | 回车键 |
| 输入多行 | Shift + 回车 |
| 打开设置 | Ctrl + O 或输入 `/config` |
| 中断回复 | Ctrl + C |
| 退出程序 | Ctrl + D 或输入 `/exit` |

---

## 内置命令

在聊天中输入以下命令：

| 命令 | 说明 |
|------|------|
| `/help` | 显示帮助信息 |
| `/config` | 打开配置编辑器 |
| `/clear` | 清除当前对话历史 |
| `/new` | 开始新对话 |
| `/exit` | 退出程序 |

---

## 配置编辑器

按 `Ctrl+O` 打开配置界面，可以：

- 查看当前配置
- 修改 AI 模型
- 设置 API 密钥
- 调整其他参数

修改后按 `q` 保存并返回聊天。

---

## 发送单条消息

不想进入聊天界面，只想快速问一个问题：

```bash
uv run tinybot agent -m "你的问题"
```

AI 会回复后自动退出。

---

## 任务进度显示

当 AI 执行复杂任务时，会显示进度：

```
任务进度：60%
正在执行：读取文件
已完成：扫描目录、分析结构
待执行：生成报告、保存文件
```

---

## 常见问题

### 输入中文乱码？

确保终端支持 UTF-8 编码。Windows 用户建议使用 PowerShell 或 Windows Terminal。

### AI 响应很慢？

可能是网络问题或模型响应慢。可以尝试更换模型或检查网络。

### 如何中断 AI 回复？

按 `Ctrl + C` 即可中断当前的 AI 回复。
