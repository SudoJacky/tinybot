Tinybot 使用教程

项目功能详细说明

返回聊天

1 快速开始

安装与初始化

使用 uv 进行依赖管理和安装：

# 安装依赖 uv sync # 初始化配置（交互式向导） uv run tinybot onboard

启动模式

命令 描述 uv run tinybot agent 交互聊天模式 uv run tinybot gateway 启动网关（WebUI + 多频道） uv run tinybot api 作为 OpenAI 兼容 API 服务运行

2 WebUI 使用指南

启用 WebUI

编辑配置文件启用 WebSocket 频道：

{ "channels": { "websocket": { "enabled": true, "host": "127.0.0.1", "port": 18790 } } }

启动 Gateway 后访问 http://127.0.0.1:18790

WebUI 功能

  • 实时流式对话，支持 Markdown 渲染和代码高亮
  • 会话管理：新建、切换、清空、删除会话
  • 工具和技能查看与管理
  • 知识库管理：添加文档、查询、重建索引
  • 工作区文件编辑
  • 配置管理：直接在界面修改所有设置
  • 中英文双语支持
  • 亮色/暗色主题切换

3 核心功能

NEW Agentic DAG 任务调度

自动将复杂任务分解为可执行的子任务 DAG：

  • 智能分解 — LLM 自动分析任务，生成带依赖关系的子任务图
  • 自动链式执行 — SubAgent 完成后自动触发依赖任务
  • 并行执行 — 并行安全的任务同时运行，最大化效率
  • 动态调整 — 运行中可添加/移除子任务

经验自进化系统

持续从问题解决经验中学习的自学习系统：

  • 语义经验搜索 — 向量搜索理解问题意图
  • 自动上下文注入 — 相关历史解决方案自动出现
  • 主动错误诊断 — 工具失败时自动触发已解决经验的建议
  • 智能置信度模型 — 多维评分评估经验有效性

NEW 知识库 (RAG)

基于向量存储的知识检索系统：

  • 支持混合检索（dense + sparse/BM25）
  • 可选 Rerank 重排序提高检索精度
  • 自动检索模式：在对话中自动注入相关知识
  • 本地 Embedding 支持，无需外部 API

Skills 技能系统

通过 Markdown 文件定义自定义技能，无需编码：

  • 技能自动加载，条件匹配时 Agent 遵循定义的工作流
  • 可在 WebUI 中创建、编辑、启用/禁用技能
  • 支持验证技能格式

4 工具系统

内置工具

工具 描述 read/write 文件读写操作 exec Shell 命令执行 web_search 网络搜索（DuckDuckGo/Brave/Tavily等） browser 浏览器自动化（通过 OpenCLI） cron 定时任务管理 mcp MCP 协议工具集成

5 CLI 交互命令

命令 描述 /config 或 Ctrl+O 打开配置编辑器 /help 显示可用命令 /clear 清除对话历史 /new 开始新对话会话 /exit 或 :q 退出聊天
