Tinybot 使用文档

轻量级个人 AI 助手框架，集成大语言模型、多聊天平台、工具系统和自动化机制。

安装与启动

# 安装依赖 uv sync # 初始化配置 uv run tinybot onboard # 交互聊天模式 uv run tinybot agent # 启动 Gateway (WebUI) uv run tinybot gateway
命令 描述
uv run tinybot agent 交互聊天模式
uv run tinybot gateway 启动网关（WebUI + 多频道）
uv run tinybot api 作为 OpenAI 兼容 API 服务
