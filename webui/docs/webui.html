WebUI 指南

网页界面

启动网页界面

运行以下命令启动：

uv run tinybot gateway

然后在浏览器打开 http://127.0.0.1:18790

界面介绍

网页界面分为几个区域：

区域 功能
左侧 会话列表、历史记录
中间 聊天对话区域
右侧 工具、知识库、技能面板

基本使用

发送消息

在底部输入框输入你的问题，点击发送或按回车键。AI 会实时回复。

新建对话

点击左侧的"新建会话"按钮开始新的对话。

查看历史

左侧会话列表显示所有历史对话，点击即可继续之前的对话。

侧边功能面板

右侧面板提供多个功能：

工具面板

查看 AI 可以使用的所有工具，比如搜索网页、读取文件等。

知识库面板

管理你的知识文档：

  • 添加新文档
  • 查看已有文档
  • 测试检索效果

技能面板

查看和管理 AI 的技能：

  • 查看可用技能列表
  • 启用或禁用技能

设置面板

修改配置，包括：

  • 更换 AI 模型
  • 设置 API 密钥
  • 调整其他参数

主题和语言

切换主题

点击右下角按钮切换亮色/暗色主题。

切换语言

点击语言按钮切换中文/英文界面。

常见问题

无法访问网页？

确保已启动 gateway 服务，且端口没有被其他程序占用。

AI 不回复？

检查是否正确配置了 API 密钥。可在设置面板查看。

如何更换模型？

在设置面板的 Agent 配置中修改 model 参数。
