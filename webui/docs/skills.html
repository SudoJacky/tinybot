技能系统

什么是技能？

技能是预设的行为模板，让 AI 在特定场景下按照特定方式工作。

示例：
  • 你要求总结文章 → AI 使用"总结技能"按要点整理
  • 你要求查天气 → AI 使用"天气技能"获取天气信息
  • 你要求定时提醒 → AI 使用"定时技能"设置任务

---

默认技能

Tinybot 内置多种技能：

| 技能 | 功能 | |------|------| | 文本总结 | 按要点整理长文本 | | 天气查询 | 获取天气信息 | | 定时任务 | 设置定时提醒 | | 记忆管理 | 管理对话记忆 | | 浏览器操作 | 自动化浏览器任务 |

---

技能触发

AI 会自动判断何时使用技能：

用户：帮我总结这篇文章

AI：[识别需要总结技能]
[按照技能模板执行]

不需要特别指令，AI 会自动匹配。

---

查看技能

网页界面

右侧技能面板显示所有可用技能：

  • 技能名称和描述
  • 启用状态开关

启用/禁用

可以关闭不需要的技能：

  • 点击开关切换状态
  • 禁用的技能不会被使用

创建自定义技能

如果你想让 AI 在特定场景有特定行为，可以创建新技能。

技能文件格式

技能是一个 Markdown 文件，包含：

---
name: my-skill
description: 我的自定义技能
---

技能说明


##触发条件
什么时候使用这个技能

    技能引用

    技能可以引用外部文件作为参考：

    ---
name: api-guide
description: API 使用指南
---

    参见 [api-docs.md] 获取完整 API 说明。

    引用的文件会自动包含在技能上下文中。

    ---

    常见问题

    技能没有触发？

    检查：

    1. 技能是否已启用
    2. 描述是否匹配你的需求
    3. 触发条件是否清晰

      4. 如何修改技能？

      直接编辑技能文件，修改后重新加载。
